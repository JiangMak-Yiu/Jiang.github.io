完美解码 官方版&绿色版

Jiang-Mak Yiu2024-07-272024-07-28

完美解码是一款能实现流行音视频、高清及4K等多媒文件软硬解播放的影音解码包，集成多媒体播放器PotPlayer、MPC-HC、MPC-BE，分离器解码器套件LAV Filters，高画质渲染器madVR，支持中英文语言平台下安装，是高清爱好者的理想播放工具。

完美解码是一款能实现各种流行视频、HDTV播放及编码的全能型影音解码包，自带 Media Player Classic、KMPlayer、PotPlayer 三款流行播放器，支持简、英语言平台下安装，能播放AVI、VCD、DVD、MPG、MP4、RMVB、TS、TP、EVO、M2TS、MKV、OGM、 MOV、SCM、CSF、FLV等众多种格式的影音文件。

软件截图

版本特色

此绿色版是基于完美解码 PureCodec 20240430 官方原版制作而成，为绿色自解压程序，可解压到桌面或磁盘任意目录（默认为 D:\Program Files\PureCodec），双击运行 “PureCodec!)绿化.cmd”，点击桌面图标 “完美解码设置” 或点击 Pure Code 文件夹中的 PureSet.exe，设置选择相应自身集成的 PotPlayer、MPC-BE、MPC-HC 播放器之一和选择相应的设置项目后，点击“应用”，然后点击桌面的“完美解码”图标后，打开需要播放的视频文件即可。

如果点击桌面的“完美解码设置”图标或点击PureCode文件夹中的PureSet.exe,设置选择相应自身集成的 PotPlayer、MPC-BE、MPC-HC 和相应的设置项目后，点击“应用”，桌面没有生成 “完美解码” 图标，
可点击 “重置” 更换播放器后选择相应的设置项目，点击 “应用” 即可在桌面生成 “完美解码” 图标。
